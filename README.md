# 카페 자동 업로드 프로그램 사용법

이 프로그램은 지정된 폴더(`contents`)에 이미지를 넣으면, 자동으로 네이버/다음 카페에 1분마다 업로드합니다.

## 1. 초기 설정 (한 번만 수행)

터미널을 열고 다음 명령어를 실행하여 필요한 라이브러리를 설치합니다. (이미 완료되었다면 건너뛰세요)

```bash
# 가상환경 활성화 (필요시)
source venv/bin/activate

# 라이브러리 설치
pip install -r requirements.txt
playwright install chromium
```

## 2. 카페 설정하기

`config.py` 파일을 열어 다음 내용을 수정하세요.

- **네이버 카페**: `NAVER` 항목의 `CAFE_ID`, `MENU_ID` 수정
- **다음 카페**: `DAUM` 항목의 `CAFE_NAME`, `GRPID`, `FLDID` 수정
- **제목**: `POST_TITLE` 수정 (기본값: "홍보 이미지 자료입니다.")

> **Tip**: 카페 ID나 게시판 ID를 모를 경우, 웹브라우저로 해당 게시판 글쓰기 화면에 들어간 뒤 URL을 확인하세요.

## 3. 최초 로그인 (필수)

자동화를 위해 브라우저에 로그인 정보를 저장해야 합니다.

1.  터미널에서 `python3 manual_login.py` 실행
2.  `1`번 선택 -> 네이버 창이 뜨면 로그인 -> 완료 후 터미널에서 엔터
3.  다시 실행 -> `2`번 선택 -> 다음(카카오) 창이 뜨면 로그인 -> 완료 후 터미널에서 엔터

이제 로그인 정보가 `user_data` 폴더에 저장되어, 이후에는 자동 로그인됩니다.

## 4. 프로그램 실행

```bash
python3 main.py
```

프로그램이 실행되면 `contents` 폴더를 계속 감시합니다.

## 5. 사용 방법

1.  `contents` 폴더에 홍보 **이미지 파일**(.jpg, .png 등)을 넣으세요.
2.  약 1분 뒤, 프로그램이 자동으로 감지하여 네이버/다음 카페에 업로드합니다.
3.  성공적으로 업로드되면 이미지는 `uploaded` 폴더로 이동됩니다.
